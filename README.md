# Cellular Vital Hub

A secure, real-time dashboard for monitoring vital signs data from Arduino devices with cellular connectivity.

## Deployment Guide

### Prerequisites
- Node.js 16+ and npm
- API key for the GSM808 sensor platform

### Environment Setup
1. Clone the repository
2. Copy `.env.example` to `.env` and fill in your configuration values:
- Arduino device with GSM808 module
- API access credentials

### Environment Variables

Create a `.env` file in the root directory with the following variables:
- Real-time vital signs monitoring (heart rate, SpO2, temperature)
- Secure API integration with Arduino GSM808 devices
- Interactive charts with historical data
- Responsive design for all devices
- Offline support with service worker
- Secure authentication system
